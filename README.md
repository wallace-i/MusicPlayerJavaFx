## TeaSound <br>
A light-weight music player and library assistant by Ian Wallace. <br>
<br>
<br>
## Keyboard Shortcuts: <br>
Play/Pause      Space<br>
Next Track      Right Arrow<br>
Previous Track  Left Arrow<br>
Stop            Ctrl + Space<br>
<br>
Vol Up          Minus<br>
Vol Down        Plus<br>
Mute            0<br>
<br>
AutoPlay        B<br>
Shuffle         N<br>
Repeat          M<br>
